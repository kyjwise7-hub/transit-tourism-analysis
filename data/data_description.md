# 데이터 설명
각 데이터를 모두 취합하여 17개의 지자체 별로 그 평균을 정리한 데이터셋을 이용하였습니다 (SPSS 이용)

## 데이터 출처
- 한국관광공사 데이터랩
- 한국교통안전공단
- 교통카드 빅데이터 시스템

## 기준연도
2023년

## 개체
17개 광역지자체

## 변수 구성

### 관광 변수 (종속)
| 변수명 | 설명 | 변환 |
|--------|------|------|
| 방문자수 | 관광 목적으로 일정시간 이상 체류한 외래관광객 수 (연단위) | 로그변환 |
| 소비액 | 해외발행 카드 결제 금액 (연단위) | 로그변환 |
| 평균숙박기간 | 1회 방문당 평균 숙박일수 | 제곱근변환 |

### 교통 지표 (독립)
| 변수명 | 설명 | 변환 |
|--------|------|------|
| 공급도 | 시가화 면적 대비 정류장 수 비율 | 로그변환 |
| 평균요금 | 1회 탑승 대중교통 요금 | - |
| 접근소요시간(분) | 최초 대중교통 이용까지 걸리는 시간 | - |
| 환승대기시간(분) | 다음 환승까지 평균 대기시간 | - |
| 환승이동시간(분) | 다음 환승까지 평균 이동시간 | - |
